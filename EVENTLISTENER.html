JavaScript Event Listener Lesson

What is an Event Listener?

An event listener in JavaScript is a method that waits for an event (like a click, keypress, or mouse movement) to occur on an HTML element. When the specified event occurs, the event listener calls a function to handle the event.

Basic Syntax of Event Listeners

The basic syntax for using an event listener is:

element.addEventListener('event', function, useCapture);

Example 1: Handling a Click Event

let button = document.getElementById('clickButton');
button.addEventListener('click', function() {
    alert('Button was clicked!');
});

Example 2: Handling a Mouseover Event

Hover over me!
let hoverDiv = document.getElementById('hoverDiv');
hoverDiv.addEventListener('mouseover', function() {
    hoverDiv.style.backgroundColor = 'orange';
});

Example 3: Handling a Keydown Event

let inputField = document.getElementById('inputField');
inputField.addEventListener('keydown', function(event) {
    console.log('Key pressed: ' + event.key);
});
