What are Logical Statements?

A logical statement is a statement or expression that evaluates to either true or false. They are typically used to make decisions in programming. In JavaScript, logical operators are used to combine or evaluate conditions in logical statements. Logical statements are essential in programming because they allow you to make decisions based on certain conditions. For example, you might want to check if a number is greater than 10, or if a user is logged in before allowing them to access a certain page.



Common Logical Operators in JavaScript



AND (&&):

This operator checks if both conditions are true. If both conditions are true, the result is true. If either one is false, the result is false.


let a = 5;
let b = 10;

console.log(a > 3 && b < 20); // true (both conditions are true)
console.log(a > 3 && b > 20); // false (second condition is false)


NOT (!):

This operator inverts the value of a condition. If the condition is true, it becomes false. If the condition is false, it becomes true.


let a = true;

console.log(!a); // false (inverts true to false)
console.log(!(a && false)); // true (inverts the false condition)


Logical Statements in if Statements

Logical statements are often used in conditional statements (like if statements) to make decisions in the code. Here's an example:


      let age = 18;
      let hasPermission = true;
      
      // Check if someone can enter the club
      if (age >= 18 && hasPermission) {
          console.log("You can enter the club!");
      } else {
          console.log("You cannot enter the club.");
      }
