Variables

store data in JavaScript and can be declared using var, let, or const. While var is function-scoped, let and const provide block-scoping, enhancing security and efficiency. 


      let a = 1
      let b = 2

Operators

perform calculations and logical operations. These include arithmetic operators (+, -, *, /), comparison operators (==, ===, !=, >, <), logical operators (&&, ||, !), and assignment operators (=, +=, -=, etc.), enabling efficient computation and decision-making. EXAMPLE:


ARITHMETIC

Arithmetic operators are used to perform mathematical operations on numbers.

EXAMPLES: 


let a = 5, b = 3;
console.log(a + b); // 8
console.log(a - b); // 2
console.log(a * b); // 15
console.log(a / b); // 1.666...
console.log(a % b); // 2
console.log(2 ** 3); // 8

Assignment Operators

Assignment operators are used to assign values to variables.


let a = 10;
a += 5;  // a = a + 5; a becomes 15
a *= 2;  // a = a * 2; a becomes 30

Comparison Operators

Comparison operators are used to compare values and return a Boolean (true or false).


let a = 5, b = 10;
console.log(a == 5);   // true
console.log(a === "5"); // false
console.log(b > a);    // true

Logical Operators

Logical operators are used to combine multiple conditions (often in if statements).


let a = true, b = false;
console.log(a && b); // false
console.log(a || b); // true
console.log(!a);     // false

Ternary (Conditional) Operator

The ternary operator is a shorthand way to write an if/else statement. 


syntax:
condition ? expressionIfTrue : expressionIfFalse;

example:
let age = 18;
let canVote = (age >= 18) ? "Yes" : "No";
console.log(canVote); // "Yes"
