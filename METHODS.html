JavaScript Built-In Methods

What Are Built-In Methods?

Built-in methods are pre-defined functions in JavaScript that allow you to manipulate and interact with different data types (such as arrays, strings, numbers, etc.), or with the DOM (Document Object Model).

Common Built-In Methods

Array Methods

push(): Adds one or more elements to the end of an array.

let arr = [1, 2, 3];
arr.push(4);  // Output: [1, 2, 3, 4]

pop(): Removes the last element from an array.

let arr = [1, 2, 3];
arr.pop();  // Output: [1, 2]

shift(): Removes the first element of an array.

let arr = [1, 2, 3];
arr.shift();  // Output: [2, 3]

unshift(): Adds one or more elements to the beginning of an array.

let arr = [1, 2];
arr.unshift(0);  // Output: [0, 1, 2]

map(): Creates a new array with the results of calling a function on each element of the original array.

let arr = [1, 2, 3];
let doubled = arr.map(x => x * 2);  // Output: [2, 4, 6]

filter(): Creates a new array with all elements that pass the test implemented by the provided function.

let arr = [1, 2, 3, 4];
let even = arr.filter(x => x % 2 === 0);  // Output: [2, 4]

String Methods

length: Returns the number of characters in a string.

let str = "hello";
console.log(str.length);  // Output: 5

toUpperCase(): Converts a string to uppercase.

let str = "hello";
console.log(str.toUpperCase());  // Output: "HELLO"

toLowerCase(): Converts a string to lowercase.

let str = "HELLO";
console.log(str.toLowerCase());  // Output: "hello"

substring(): Extracts a section of the string.

let str = "hello world";
console.log(str.substring(0, 5));  // Output: "hello"

includes(): Checks if a string contains a specific substring.

let str = "hello world";
console.log(str.includes("world"));  // Output: true

Number Methods

toFixed(): Formats a number to a fixed number of decimals.

let num = 3.14159;
console.log(num.toFixed(2));  // Output: "3.14"

parseInt(): Converts a string to an integer.

let num = "42";
console.log(parseInt(num));  // Output: 42

parseFloat(): Converts a string to a floating-point number.

let num = "3.14";
console.log(parseFloat(num));  // Output: 3.14

Object Methods

Object.keys(): Returns an array of a given object's property names.

let obj = { name: "John", age: 30 };
console.log(Object.keys(obj));  // Output: ["name", "age"]

Object.values(): Returns an array of a given object's property values.

let obj = { name: "John", age: 30 };
console.log(Object.values(obj));  // Output: ["John", 30]

Object.assign(): Copies all enumerable properties from one or more source objects to a target object.

let target = { name: "John" };
let source = { age: 30 };
let result = Object.assign(target, source);
console.log(result);  // Output: { name: "John", age: 30 }

DOM Methods

getElementById(): Selects an element by its `id`.

let element = document.getElementById("myElement");

querySelector(): Selects the first element that matches a CSS selector.

let element = document.querySelector(".myClass");

addEventListener(): Attaches an event listener to an element.

let button = document.querySelector("button");
button.addEventListener("click", function() {
  alert("Button clicked!");
});

Conclusion

These are just a few examples of JavaScript's built-in methods. They provide powerful functionality to manipulate data types and the DOM, making development easier and faster.
